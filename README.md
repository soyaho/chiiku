# てんびん（chiiku）

6歳児（年長）向け知育ミニゲーム群の**第1号かつ触感の基準器**。
画面中央の天秤に、散らばったりんごを指でつまんで載せていく。全部載せ終わると、
下がった側（＝多い側）の動物が喜んで食べる——それが答え。

学習対象は数量比較の身体感覚（**多い＝重い＝下がる**）。

## 設計原則（このゲームが検証するもの）

- **動詞が先** — プレイヤーは選択肢を選ばない。ドラッグの結果として答えが世界に現れる
- **世界が答える** — 正誤判定UIを作らない。天秤の傾きという物理現象が答えを返す
- **✗を出さない** — 失敗演出・ブザー・バツ印は全面禁止。やり直しは常に静かに可能

仕様書: [`docs/tenbin_spec_v0.1.md`](docs/tenbin_spec_v0.1.md)

## 遊び方・動かし方

```bash
npx http-server -p 8080 .
# → http://localhost:8080/index.html （iPad Safari 横持ち想定。マウスでも遊べる）
```

- 成果物は `index.html` **1ファイルのみ**（外部依存・アセット・ビルドなし）
- 1セッション5問。5問目を終えると星のご褒美画面、タップでもう1回
- 画面左上隅を**2秒長押し**で親メニュー（プレイログの確認・コピー・消去）
- プレイログは `localStorage` キー `tenbin_logs` に蓄積（スキーマは仕様書 §6）

## 検証

```bash
node harness/run.js
```

Playwright＋実イベント（実インターフェース主義）の E2E 一式。テストフックの契約は
[`docs/test-hooks.md`](docs/test-hooks.md)（読み取り専用 `window.__tenbin`、注入は `?seed` / `?timescale` のみ）。
