# てんびん — 基準プロトタイプ仕様書 v0.1

## 0. このプロジェクトの位置づけ

6歳児(年長)向け知育ミニゲーム群の**第1号かつ触感の基準器**。
本作の目的はゲーム1本の完成ではなく、以下の設計原則がデジタルで成立するかの検証にある。

- **動詞が先**: プレイヤーは選択肢を選ばない。操作(ドラッグ)の結果として答えが世界に現れる
- **世界が答える**: 正誤判定UIを作らない。天秤の傾きという物理現象が答えを返す
- **✗を出さない**: 失敗演出・ブザー音・バツ印は全面禁止。やり直しは常に静かに可能

ここで作るドラッグ物理・スナップ・効果音・パーティクル・ログの各部品は、後続ミニゲームの共通シェルに昇格させる前提。ただし v0.1 では過剰な抽象化はせず、分離しやすいコード構造(セクションコメント区切り)に留めること。

## 1. 成果物・技術要件

| 項目 | 要件 |
|---|---|
| 成果物 | `index.html` **1ファイルのみ**(外部依存・外部アセット・ビルド工程なし) |
| 描画 | Canvas 2D、`requestAnimationFrame`、60fps目標 |
| 解像度 | `devicePixelRatio` 対応。論理座標系は 1024×768 固定でスケーリング |
| 入力 | タッチイベント(`touchstart/move/end`)+マウスフォールバック。マルチタッチは最初の指のみ有効 |
| 音 | **WebAudio APIで全音を合成生成**(音声ファイル不要)。初回タッチで `AudioContext` を初期化 |
| 実行環境 | iPad Safari。ホーム画面追加で全画面化(`apple-mobile-web-app-capable` 等のmeta必須) |
| 画面向き | 横持ち前提。縦持ち検出時は「回して」を示す絵(文字なし)を表示 |
| 文字 | **画面内の文字表示は全面禁止**。数字も原則禁止(数はドット・個数表現で示す)。例外は親メニュー内のみ |

## 2. ゲーム概要

画面中央に大きな天秤。左右の空間にりんごが散らばっている。
子どもはりんごを指でつまんで天秤の皿に載せていく。載せるたびに天秤がぎいっと傾く。
全部載せ終わると、**下がった側(=多い側)にいる動物が喜んで食べる**。それが答え。

学習対象: 数量比較の身体感覚(多い = 重い = 下がる)。

---

## 3. 画面構成(論理座標 1024×768)

```
+----------------------------------------------------------+
|  ○○○○○ (進行ドット: 現在の問題位置。上端中央・小さく)      |
|                                                          |
|   りんご散布エリア(左)         りんご散布エリア(右)       |
|      🍎 🍎                         🍎 🍎 🍎              |
|        🍎                           🍎                   |
|                                                          |
|        [左の皿]      ▲支点      [右の皿]                 |
|            \________|________/                           |
|   🐻(左の動物)     支柱      🐰(右の動物)                |
+----------------------------------------------------------+
```

- 左右のりんごは色または模様で薄くグループ分けしてよいが、皿への載せ先は制限しない(左のりんごを右皿に載せても物理としては正しく動く)
- 動物は単純図形の組み合わせで描く(クマ=茶の円組み合わせ、ウサギ=白+耳、程度で可)

## 4. インタラクション仕様

### 4.1 りんごのドラッグ

- タッチ判定半径: 見た目より大きく **60px以上**(6歳の指精度対策)
- つまむと少し拡大(1.15倍)+「ぽっ」という音
- ドラッグ中は指に追従(慣性つき: 位置をlerp係数0.4程度で追う)
- 皿の上空(皿中心から半径120px)で放す → 皿に**スナップして着地**、「ことん」という音
- 皿以外で放す → 重力で落下し、地面でバウンドして静止(その場に残る。ペナルティなし)
- 皿に載ったりんごは皿の子要素になり、皿の傾き・上下動に追従する。皿上での配置は自動整列(2〜3個ずつ積む)
- 皿からりんごをつまんで下ろすことも可能(可逆)

### 4.2 天秤の物理

- 傾き目標角 `θ_target = clamp((右個数 − 左個数) × 6°, −30°, +30°)`
- 実際の角度はバネ-ダンパで追従: `θ'' = k(θ_target − θ) − c·θ'`
  - **臨界減衰にしない**。1〜2回ゆらっと揺れてから静止する係数にチューニング(この揺れが触感の核)
- りんごが載る瞬間、その側が一瞬沈み込むオーバーシュートを入れる
- 傾き変化中は「ぎぃ」という軋み音(角速度に応じて音量変化)
- 皿は腕の端に吊り下げ表現(常に水平を保つ)

### 4.3 問題の完了判定

- 場のりんごが全て皿に載った状態で天秤が静止(角速度≒0が1秒継続)したら完了
- 完了演出:
  - **差がある場合**: 下がった側の動物がぴょんと跳ね、皿のりんごを1個ずつ食べる(食べるたび「ぱくっ」)。食べ終わると星パーティクル+短いファンファーレ(合成音の明るいアルペジオ)
  - **同数(釣り合い)の場合**: 天秤が水平でゆらゆら静止 → 両側の動物が同時に跳ねて半分ずつ食べる。**釣り合いは特別に美しい演出**(虹色パーティクル等)にする。同数=つまらない、ではなく同数=特別、と感じさせる
- 演出後1.5秒で次の問題へ自動遷移(フェード)

### 4.4 禁止事項(再掲・厳守)

- ✗印、ブザー、失敗アニメ、減点表現
- 制限時間、カウントダウン
- 文字・数字の表示(ドット表現は可)
- 点滅の激しい演出(1秒3回以上の明滅禁止)

## 5. セッション構成とレベルテーブル

1セッション = 5問。データ駆動で以下のテーブルを定義し、生成ロジックはテーブルを読むだけにする(後続ゲームでの流用を想定)。

```js
const LEVELS = [
  { left: 2, right: 5, scatter: 0.2 },  // 明確な差・ほぼ整列
  { left: 4, right: 3, scatter: 0.3 },  // 差1・整列気味
  { left: 5, right: 4, scatter: 0.7 },  // 差1・散らばり(ぱっと見で数えにくく)
  { left: 5, right: 5, scatter: 0.6 },  // 同数 → 釣り合い体験
  { left: 7, right: 6, scatter: 0.8 },  // 数が増える・散らばり
];
```

- `scatter`: 0=格子整列、1=完全ランダム配置(重なりは回避)
- 左右どちらが多いかは問題ごとにランダムで反転させる
- 5問完了でセッション終了画面: 星が5つ並んで輝く+動物たちが手を振る。タップでもう1回(セッション再開)

## 6. ログ仕様(淘汰判定用データ)

`localStorage` キー `tenbin_logs` に配列でセッション単位で保存。

```js
{
  sessionStart: <ISO文字列>,
  completed: <bool>,             // 5問完走したか
  retried: <bool>,               // 終了画面で「もう1回」を押したか
  problems: [
    {
      level: <index>,
      firstTouchMs: <数値>,      // 問題表示→最初にりんごに触れるまで
      durationMs: <数値>,        // 問題表示→完了まで
      dropOutside: <数値>,       // 皿の外で放した回数
      removedFromPlate: <数値>   // 皿から下ろし直した回数
    }
  ]
}
```

### 親メニュー

- 画面左上隅を**2秒長押し**で開く(子どもが偶発的に開かないため)
- 内容: セッション数・完走率・もう1回率の簡易表示、ログJSONのクリップボードコピー、ログ全消去
- ここのみ文字使用可(日本語)

## 7. アート・音の方向性

- 背景: 明るい単色〜2色のやわらかいグラデーション。要素数を絞り、余白を広く取る
- 全オブジェクトはCanvasプリミティブ(円・角丸矩形・ベジェ)で描画。絵文字は使わない
- アニメーションはease系(easeOutBack等)で「もちっ」とした質感を統一
- 音はすべてWebAudioのオシレータ+エンベロープで合成:
  - つまむ「ぽっ」: sine 短音・ピッチ上昇
  - 載せる「ことん」: 低めのsine+短いノイズ
  - 軋み「ぎぃ」: 低周波sawをローパスで柔らかく・角速度連動
  - 食べる「ぱくっ」: 短いノイズバースト
  - ファンファーレ: メジャーアルペジオ3〜5音
